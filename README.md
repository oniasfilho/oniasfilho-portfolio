## Overview

This project is a modern full-stack portfolio application built to
showcase my work as a Software Engineer, with a strong focus on:

-   SSR (Server-Side Rendering)
-   Scalable UI systems
-   Clean separation between presentation and business logic

------------------------------------------------------------------------

## Tech Stack

### Frontend

-   Next.js (App Router)
-   React
-   TypeScript
-   CSS

### Testing

-   Vitest

### Deployment

-   Vercel

## Running Locally

``` bash
git clone https://github.com/oniasfilho/oniasfilho-portfolio.git
cd oniasfilho-portfolio
npm install
npm run dev
```
